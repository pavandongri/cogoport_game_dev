# cogoport_game_dev

## steps to run the game
1. clone the project to local machine
2. open the project in p5.js editor
3. run the code



## steps to play the game
1. initially you will see a blue window with score text at the top left of banner
2. press 'ENTER KEY' on the keyboard to start the game
3. once you press 'ENTER KEY' the game starts.
4. use left arrow key to move the paddle left side ... use right arrow key to move the paddle right side
5. if the ball falls at the bottom of banner .. the game ends
6. if we break all the bricks without ball fall down.. we will win the game



## NOTE:  To start the game press "ENTER KEY' in the keyboard
